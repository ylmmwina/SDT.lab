Detailed Description

template<typename TNode, typename TEdge>
class Graph< TNode, TEdge >

Шаблонний клас, що представляє граф зі списком суміжності.

Граф може бути орієнтованим або неорієнтованим. Використовує std::map для зберігання вершин та їхніх сусідів.

Template Parameters
TNodeТип даних для вершин (наприклад, int, std::string).
TEdgeТип даних для даних/ваги ребер.
